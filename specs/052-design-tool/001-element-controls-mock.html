052 · 001 — Element Controls

Controls around a selected or inserted element: text edit, resize, drag/move, and delete. Text moves from the bottom toolbar to per-element controls. Exploring multiple treatments for how controls integrate with the existing teal selection outline.

Option A — Outline-Integrated

Controls sit directly on the selection outline — no extra background chrome. Teal circles on the outline, feel like part of the selection frame itself. Drag grip top-left, text top-right, delete top-right corner outside. Resize: right edge, bottom edge, bottom-right corner only.

Option C — Outline-Embedded (controls in the border)

Insertion Point

The insertion point is a horizontal line with circle endpoints — not a box. No drag, no delete at this stage. Just the line marking where new content will go. The text button sits on the line so the user can insert typed text at that position. Clicking text inserts a placeholder element and enters edit mode.
