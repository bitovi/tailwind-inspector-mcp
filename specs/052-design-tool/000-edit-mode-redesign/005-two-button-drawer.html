Interaction Flow
1. Click element → State A: teal outline + drawer with [Edit text] [Describe change]
2a. Click "Describe change" → State B: msg row appears, type description, send. ← back returns to A.
2b. Click "Edit text" → State C: outline→orange, text node highlighted, inline editing. Discard/Done in header. ← back = discard + return to A.
3. "Queue" → Change queued. Returns to State A.
Escape / ← back → Always returns to previous state (C→A, B→A, A→deselect).
