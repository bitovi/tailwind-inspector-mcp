Option A
"+" button per section → dropdown of available prefixes
Click the [+] in the section header → a dropdown lists the properties you can add. Pick one → a blank scrubber/color picker appears. Two-step but very discoverable.

Option B
Inline type-to-search per section
Each section has a compact search input. Type a class name → autocomplete filters matching Tailwind classes. Fast for power users who know the class names.

Hybrid — active classes inline, ghost "add" row below
