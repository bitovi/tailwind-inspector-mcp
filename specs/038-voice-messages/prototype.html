038 · Voice Messages — Overlay Message Row

Add a microphone button to the overlay's inline message row. Uses the browser's built-in Web Speech API (SpeechRecognition / webkitSpeechRecognition) — no server-side transcription, no API keys. Interim results stream into the textarea as the user speaks. The button only renders in browsers that support the API.

Mic Button State Machine

Three states: idle (default), listening (recording + transcribing), and error (mic blocked or API failure). The button is completely hidden if the browser doesn't support SpeechRecognition.

Browser Support

The Web Speech API is available in all Chromium browsers and Safari. When unavailable, the mic button simply doesn't render — the typing experience is unchanged.

Browser API Notes
Chrome 33+ ✅ webkitSpeechRecognition Best support, most VyBit users
Edge 79+ ✅ webkitSpeechRecognition Chromium-based, same engine
Safari 14.1+ ✅ SpeechRecognition macOS and iOS
Firefox ❌ Not shipped Behind flag only — button hidden
